# Instruções para Teste

Importe o repositório para a sua IDE. 
Crie um banco de dados chamado "empresa_rh" no seu banco de dados local, utilizei o mysql para o teste.

Rode a aplicação a partir da classe RhSpringbootApplication.java

Acesse localhost:8080 e comece populando Cargos e Departamentos antes de popular os funcionários, isso é necessário por questões de integridade (funcionários não podem ter cargos e departamentos vazios)

Teste as funcionalidades do nosso incrível sistema de RH!
Crie cargos, departamentos e popule com centenas de funcionários.
